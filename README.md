# celo-clicker-sdk

TypeScript SDK for interacting with the CeloClicker contracts on Celo.

## Installation

```bash
npm install celo-clicker-sdk
```

## Usage

```ts
import { CeloClickerClient } from "celo-clicker-sdk";

const client = new CeloClickerClient({
  contractAddress: "0x..."
});
```

## Development

```bash
npm install
npm run build
```
